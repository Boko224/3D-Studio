# Структура на проекта

## Новосъздадени файлове и компоненти

### Конфигурационни файлове
- `package.json` - Зависимости (React, Vite, Tailwind, Lucide Icons)
- `vite.config.js` - Vite конфигурация
- `tailwind.config.js` - Tailwind CSS конфигурация
- `postcss.config.js` - PostCSS конфигурация
- `.gitignore` - Git ignore файл

### React приложение (`src/`)

#### Главни файлове
- `main.jsx` - Входна точка на приложението
- `App.jsx` - Главен компонент с routing

#### Компоненти (`src/components/`)
- `Navbar.jsx` - Навигационна лента с мобилна поддръжка
- `Footer.jsx` - Долна част на страницата
- `ProductCard.jsx` - Карта на продукт
- `Button.jsx` - Преизползван бутон компонент
- `Input.jsx` - Преизползван input компонент

#### Контекст (`src/context/`)
- `CartContext.jsx` - Global cart state management (AddToCart, RemoveFromCart, UpdateQuantity)

#### Данни (`src/data/`)
- `products.js` - Примерни продукти, категории, цветове, материали, методи на доставка

#### Страни (`src/pages/`)
- `Home.jsx` - Начална страница (hero, features, популярни продукти)
- `Shop.jsx` - Каталог на продукти със филтриране по категория и търсене
- `ProductDetail.jsx` - Детайли на продукт със персонализация (текст, цвят, материал)
- `Cart.jsx` - Кошница със чекаут форма за доставка
- `Upload.jsx` - Качване на 3D модели с drag & drop и форма
- `About.jsx` - За нас страница

#### Стилове
- `index.css` - Глобални Tailwind стилове и CSS класове
- `App.css` - Специфични за App стилове

### HTML & Документация
- `index.html` - Главна HTML страница
- `README.md` - Документация на проекта
- `PROJECT_STRUCTURE.md` - Описание на файловата структура (този файл)

## Функционалности

### 1. **Магазин (Shop)**
- Каталог на продукти
- Филтриране по категория (Всички, Ключодържатели, Фигурки, Части, Организатори)
- Търсене по име и описание
- Адаптивен grid на продукти

### 2. **Персонализация (Product Detail)**
- Вход на персонален текст (макс. 15 символа)
- Избор на цвят из предефинирани опции
- Избор на материал (PLA / PETG с +2 лв. за PETG)
- Управление на количество
- Динамично ценообразуване

### 3. **Качване на модели (Upload)**
- Drag & drop функционалност за качване на файлове
- Поддръжка на .STL, .OBJ, .GCODE формати
- Валидация на размер на файл (макс. 50MB)
- Форма за контакт (име, email, телефон, описание)
- FAQ раздел
- Успешно потвърждение

### 4. **Кошница (Cart)**
- Преглед на всички продукти в кошницата
- Управление на количество (увеличи/намали)
- Премахване на продукти
- Избор на куриер (Econt, Speedy, Локално теглене)
- Форма за доставка
- Итоговая цена (продукти + доставка)
- LocalStorage синхронизация

### 5. **Начална страница (Home)**
- Hero секция със CTA бутони
- Процес (4 стъпки) визуализация
- Популярни продукти
- Call-to-action за качване на модели

### 6. **За нас (About)**
- Информация за компанията
- Ценности (качество, клиентоцентричност, инновация)
- Процес на работа
- Информация за екипа
- Контактна информация

### 7. **Навигация (Navbar)**
- Логото и линкове към страни
- Икона за кошница с броячка
- Мобилна версия с hamburger меню
- Sticky navbar

### 8. **Контекст за кошница**
- LocalStorage база персистентност
- Функции: addToCart, removeFromCart, updateQuantity, clearCart, getTotalPrice
- React Context за глобално държавство

## Дизайн

### Цветова палета
- Първичен: Indigo (#4f46e5)
- Вторичен: Purple (#7c3aed)
- Фон: White/Gray-50
- Текст: Gray-900/Gray-600
- Акценти: Red (за грешки), Green (за успех)

### UI Компоненти
- Tailwind CSS за всички стилове
- Lucide React за икони
- Преизползвани Button и Input компоненти
- Адаптивен дизайн (мобилен + desktop)

## Как да стартираш

1. **Инсталирай зависимостите**
   ```bash
   npm install
   ```

2. **Стартирай dev сервера**
   ```bash
   npm run dev
   ```

3. **Отвори браузъра**
   ```
   http://localhost:5173
   ```

4. **За production**
   ```bash
   npm run build
   ```
